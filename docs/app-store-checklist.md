# App Store 上架检查清单

## 产品定位

- 定位为成人性健康、伴侣沟通、私密记录工具。
- 截图和宣传语避免露骨色情化表达。
- 年龄分级如实选择成人健康与性主题相关项。

## 必备页面

- Web 宣传页：`/`
- 隐私政策：`/privacy`
- 服务条款：`/terms`
- 支持页面：`/support`
- 删除账号与数据说明：`/delete-account`

## App 内入口

- 我的页展示隐私政策、服务条款、支持、删除账号。
- 账号删除必须能在 App 内发起或清楚说明处理方式。
- 伴侣共享必须逐项授权，不能默认公开亲密记录。

## 内容治理

- 广场和摇一摇只允许预设短语。
- 必须有举报入口。
- 必须能屏蔽或下架违规内容。
- 禁止自由文本陌生聊天作为 V1 默认能力。

## 医疗与健康声明

- 经期预测仅作估算，不构成医疗诊断。
- 性健康知识仅作教育，不替代医生建议。
- 高风险行为提示应引导用户咨询专业人士。

## 隐私

- 亲密记录默认私密。
- 支持删除账号和导出数据。
- Token 存储：iOS 使用 Keychain，Web 使用安全 Cookie 或受控本地存储策略。
- 生产环境必须使用 HTTPS。
