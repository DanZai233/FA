# 预设短语轻社交设计

法了么 V1 不开放自由文本陌生聊天。所有广场、伴侣留言、摇一摇破冰都来自预设短语拼句。设计参考的是“模板 + 分类词库 + 评价反馈”的留言机制，不在产品 UI 中直接暴露参考来源。

## 拼句结构

一句话由四个槽位组成：

- `tone`：语气，例如「安全员已上线」「嘴硬但诚实」
- `subject`：主语，例如「我的荷尔蒙」「这位成年人」
- `action`：动作，例如「申请抱抱」「提醒戴好装备」
- `ending`：收尾，例如「尊重同意最性感」「先喝水再说」

展示格式：

```text
安全员已上线 / 今日小火苗 / 提醒戴好装备 / 尊重同意最性感
```

## 场景

- 伴侣留言：更偏沟通、同意、保护、关心。
- 匿名广场：更偏自嘲、共鸣、健康提醒。
- 摇一摇：只提供低风险破冰，不展示位置，不开放自由聊天。

## 内容治理

- 预设短语必须人工审核后上线。
- 用户发布内容只保存拼句结果和模板 ID，不保存自由文本。
- 广场内容支持举报、屏蔽、限流。
- 禁止未成年人、非自愿、胁迫、交易、仇恨、露骨色情和违法暗示。

## 推荐新增短语

- 语气：`先别上头`、`成年人的理智回来了`、`今晚不硬撑`
- 主语：`安全员`、`小火苗`、`恋爱脑`
- 动作：`建议戴好装备`、`申请暂停三分钟`、`请求确认同意`
- 收尾：`不舒服就停止`、`风险别硬赌`、`温柔比速度重要`
